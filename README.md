Personal Blog Platform

A full-stack personal blogging platform built with Next.js (frontend) and Node.js/Express (backend).
This project allows users to create, edit, delete, and manage blog posts with a clean UI and a REST API.
Docker is used for containerizing both frontend and backend services.
